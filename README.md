## To Run react version

Clone the project

```bash
  https://github.com/PALASH-BAJPAI/Sample-Portfolio.git
```
And switch to reactImplementation branch

Go to the project directory

```bash
  cd App
```

Install the requirements

```bash
  npm install
```

Run your app locally

```bash
  npm start
```
